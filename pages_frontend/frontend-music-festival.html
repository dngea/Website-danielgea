Overview

A junior frontend project built using HTML, Sass, and JavaScript, compiled with Gulp. The project showcases a website for an indie music festival, featuring a simple one-page design, responsive layouts, a dynamic gallery and a few interactive elements. It provides users with information about the event, lineup, and ticket options.
